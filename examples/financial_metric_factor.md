# 财务指标因子样例

## 用户输入

使用 factor-research，研究经营现金流 / 市值这个指标有没有 alpha。

## 应识别入口

指标/公式 -> 指标设计路径。

## 关键设计

- 因子名称：经营现金流市值比因子
- 公式：TTM 经营活动现金流净额 / 总市值
- 数据频率：财报季频，交易信号可月频或季频更新
- 信号时间：财报实际披露后下一个可交易日
- 标准化：行业内分位数或 z-score
- 反证条件：现金流一次性项目驱动、周期行业口径不稳定、小市值流动性不足

## 产物重点

- 未来函数检查必须确认使用财报披露日，而不是报告期截止日。
- `factor_definition.yaml` 中 `available_at` 必须写披露后可得。
