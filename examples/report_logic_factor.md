# 研报逻辑因子样例

## 用户输入

使用 factor-research，把研报里的“高研发投入公司未来成长更好”设计成成长因子。

## 应识别入口

研报 -> 研报逻辑路径。

## 关键设计

- 因子名称：研发投入强度成长因子
- 核心变量：研发费用 / 营业收入
- 可选变量：研发费用同比增速、研发人员占比、研发资本化比例
- 标准化：行业内 z-score
- 反证条件：高研发但收入转化低、研发资本化比例异常高、现金流恶化

## 产物重点

- `source_notes.md` 记录机构、报告标题、发布日期、分析师。
- `FACTOR_DESIGN.md` 必须区分可量化观点和不可量化背景。
- 财务数据必须处理披露滞后。
